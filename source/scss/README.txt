SETTINGS - Global variables, config switches, buttons, colors

ex. font family, font size, font color etc.

TOOLS - Default mixins and functions

GENERIC - Ground zero styles

ex. Normalize.css, resets, box-sizing

BASE - Unclassed HTML elements. Uses HTML elements

ex. ul, a, h1, h2

OBJECTS - Cosmetic free design patterns. Uses classes

ex. margin, padding, position

COMPONENTS - Design pieces of UI. Uses classes

TRUMPS - Overrides, helpers, utilities. Usually with !important
